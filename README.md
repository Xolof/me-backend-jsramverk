## Installation
```
npm install
```

## Run the application

Set an environment variable in your terminal to some long and random string.
```
export JWT_SECRET='ThisIsJustAnExampleItShouldBeLongAndRandom'
```

Then you can start the applicaton.

```
npm start
```
